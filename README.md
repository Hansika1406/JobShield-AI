# JobShield AI 🛡️

JobShield AI is an AI-powered fake internship and job offer detection system built using Machine Learning and Natural Language Processing (NLP).

## Features

* Fake vs Legitimate Job Detection
* Fraud Risk Scoring
* PDF & TXT Job Offer Analysis
* Scam Pattern Detection
* WhatsApp & Payment Scam Detection
* Recruiter Email Verification
* AI Risk Summary
* Analysis History Dashboard

## Tech Stack

* Python
* Machine Learning
* NLP (Natural Language Processing)
* Streamlit
* Scikit-Learn
* Pandas
* PyPDF2

## Problem Statement

Many students fall victim to fake internships and job scams involving registration fees, fake recruiter emails, unrealistic salaries, and misleading hiring tactics. JobShield AI helps identify such fraudulent opportunities.

## Future Scope

* Real-time job URL scanning
* Company verification API
* Resume-job compatibility analysis
